for循环

  • 什么时候使用for循环？
  • 当我们需要同时处理一组数据得时候，就需要使用循环
  • 需要什么？
  1. 起始值
  2. 判断条件
  3. 执行语句
  4. 自增
  • 写法：
  • 当变量的判断条件永远成立得时候，这个循环就会变成一个死循环。
  • 循环变量再for循环结束之后一样调用，不过调用得值是for循环不符合条件之后得值。
  • 当判断条件不成立时，循环就结束了。

this

  • 在事件函数中this代表触发当前事件的元素。
  • 在所有非事件函数中this指向window。
  • 谁调用这个函数，this就指向谁。
  • 函数执行的时候都属于window调用
